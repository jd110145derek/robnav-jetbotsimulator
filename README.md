# robnav-jetbotsimulator

## Following the steps:


1. Run Server
* cd Message-Server
* ./DataServer

2. Run Simulator
* cd Unity-Simulator
* sh run.sh

3. Run Python Code
* cd Python-Wrapper
* python example.py
